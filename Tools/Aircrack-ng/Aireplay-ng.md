# Aireplay-ng

Aireplay-ng is used to inject frames.

## Usage

Perform an injection test

* ```aireplay-ng -9 wlan1mon```

Perform a card to card injection test

* ```aireplay-ng -9 -i wlan0 wlan1mon```
