# Empire

```bash
# Installing Empire
git clone --recursive https://github.com/BC-SECURITY/Empire
cd Empire/setup
./install.sh

# Starting the Empire server
./ps-empire-server server

# Interacting with the Empire server
./ps-empire-server client
```
